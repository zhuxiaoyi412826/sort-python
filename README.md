# 排序算法可视化工具

基于 **Pygame** 实现的排序算法可视化桌面应用，支持 **19 种排序算法**的动态演示，适用于算法教学与学习。

---

## ✨ 功能特性

### 核心可视化
- **19 种排序算法**：10 种基础排序 + 9 种趣味排序
- **实时动态演示**：蓝色竖条表示数据，黄色高亮当前比较/交换，绿色表示排序完成
- **速度控制**：0.25x ~ 256x 多档速度调节
- **数据量调节**：1 ~ 1000 条数据，滑动条实时预览
- **算法代码面板**：点击"算法代码"按钮，右侧浮层展示当前算法的 Python 源码，支持语法高亮和滚动浏览

### 🎵 音效反馈系统
- **比较音效**：音高与柱子高度成正比，数据越大音调越高
- **交换音效**：清脆的叮声提示数据交换
- **完成音效**：排序完成时播放竖琴琶音
- **自定义音效**：支持加载 `avi/1.wav` 作为排序音效
- **独立控制**：过程音效和完成音效可分别开关

### 🎨 个性化设置面板
点击右上角 ⚙ 设置按钮打开设置面板：
- **颜色自定义**：7 种常见颜色（红橙黄绿蓝紫青）下拉框选择，分别设置柱状图/排序中/已完成颜色
- **音效开关**：独立控制过程音效和完成音效
- **亮度调节**：20%~100% 亮度滑动条，减号/加号按钮精细调节
- **深色/浅色模式**：一键切换主题，偏好设置自动保存

### 📊 多算法对比模式
点击"多算法对比"按钮，打开独立对比窗口：
- **选择 2~9 种算法**：复选框配置弹窗，实时显示已选数量
- **Matplotlib 实时图表**：每种算法独立子图，动态柱状图/折线图动画
- **复杂度信息**：每个图表标题显示算法名称 + 时间复杂度 + 空间复杂度
- **排行榜面板**：按耗时实时排序，最优绿色高亮，最差红色标注
- **倍速控制**：0.25x~8x 六档倍速调节
- **导出功能**：支持导出 PNG 图片和 JSON 数据

### 数据分布选择
悬停下拉框，支持 5 种数据分布模式，直观展示算法在不同输入下的表现：
| 分布模式 | 说明 | 教学价值 |
|----------|------|----------|
| 随机分布 | 完全随机 | 通用基准 |
| 近乎有序 | 95% 有序 + 5% 随机交换 | 展示插入排序优势 |
| 完全逆序 | 从大到小排列 | 展示冒泡排序最坏情况 |
| 少量唯一值 | 仅 5~8 种不同值重复 | 展示计数/桶排序优势 |
| 正弦波形 | 正弦曲线分布 | 展示算法对周期模式的处理 |

### 算法信息面板
控制栏下方实时显示当前算法的：
- 英文名称 + 时间复杂度 + 空间复杂度
- 排序耗时（实时计时）
- 数据规模
- 排序完成度（进度百分比）

### 状态提示
三态自动切换：`● 就绪中` → `▶ 排序中...` → `✓ 排序完成!`

### 其他功能
- 全屏模式
- 切换算法时保留当前数据（便于对比不同算法在同一输入下的表现）
- 数量设置弹窗：滑动条拖拽 + 数字输入框双模式
- 深色/浅色主题自动同步到多算法对比窗口

## 🚀 快速开始

### 环境要求
- Python 3.8+
- Pygame 2.0+
- Matplotlib 3.0+（多算法对比模式）
- Tkinter（Python 自带）

### 安装与运行

```bash
# 安装依赖
pip install pygame matplotlib

# 运行主程序
python sorting_visualizer.py

# 或直接运行多算法对比模式
python compare_mode.py
```

## 📊 支持的排序算法

### 基础排序（10 种）
| 算法 | 英文名 | 时间复杂度 | 空间复杂度 |
|------|--------|-----------|-----------|
| 冒泡排序 | Bubble Sort | O(n²) | O(1) |
| 选择排序 | Selection Sort | O(n²) | O(1) |
| 插入排序 | Insertion Sort | O(n²) | O(1) |
| 快速排序 | Quick Sort | O(n log n) | O(log n) |
| 归并排序 | Merge Sort | O(n log n) | O(n) |
| 希尔排序 | Shell Sort | O(n^1.3) | O(1) |
| 堆排序 | Heap Sort | O(n log n) | O(1) |
| 桶排序 | Bucket Sort | O(n+k) | O(n+k) |
| 计数排序 | Counting Sort | O(n+k) | O(k) |
| 基数排序 | Radix Sort | O(d·(n+k)) | O(n+k) |

### 趣味排序（9 种）
| 算法 | 英文名 | 时间复杂度 | 空间复杂度 |
|------|--------|-----------|-----------|
| 猴子排序 | Bogo Sort | O((n+1)!) | O(1) |
| 睡眠排序 | Sleep Sort | O(n+max) | O(n) |
| 面条排序 | Spaghetti Sort | O(n²) | O(n) |
| 斯大林排序 | Stalin Sort | O(n) | O(1) |
| 鸡尾酒排序 | Cocktail Sort | O(n²) | O(1) |
| 慢排序 | Slow Sort | O(n^log n) | O(log n) |
| 煎饼排序 | Pancake Sort | O(n²) | O(1) |
| 珠排序 | Bead Sort | O(n·max) | O(n·max) |
| 鸽巢排序 | Pigeonhole Sort | O(n+k) | O(n+k) |

## 🛠️ 技术栈

- **Python 3** — 主语言
- **Pygame** — 图形渲染与事件处理
- **Matplotlib** — 多算法对比图表绘制
- **Tkinter** — 多算法对比窗口 UI
- **生成器模式** — 所有排序算法均为 `yield` 生成器，逐步产出可视化状态
- **PCM 音频合成** — 纯 Python 生成拨弦/叮声音效，无需外部音频库

## 🎮 操作说明

|------|------|

| 操作 | 说明 |

| 操作 | 说明 |
